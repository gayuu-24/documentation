File Handling

File handling in Python is a process of working with files, including reading from and writing to them.

Python provides built-in functions and methods for file operations, making it easy to manage files effectively.

  • 'r': Read mode (default).
  • 'w': Write mode (overwrites file if it exists).
  • 'a': Append mode (adds content to the end).
  • 'r+': Read and write mode.
  • 'b': Binary mode (e.g., 'rb' for binary read).

Opening a File

Syntax:

file = open('example.txt', 'mode')

Reading from a File

Example:

with open('file.txt', 'r') as file:

    content = file.read()

    print(content)

This code opens the file file.txt in read mode ('r') and assigns its content to the variable content.

The with statement ensures the file is automatically closed after reading, and the print(content) displays the file's contents.

Output (In textfile):

Hello, World!

Writing to a File

Example:

with open('file.txt', 'w') as file:

   file.write("Hello!")

Output:

No visible output in the console, but the file file.txt will contain the following text:

Hello!

This code opens file.txt in write mode ('w'), creating it if it doesn't exist, and writes the text "Hello!" to it. The with statement ensures the file is automatically closed after writing.

Appending to a File

Example:

with open('file.txt', 'a') as file:

   file.write("\nAppended text.")

Output:

No visible output in the console, but the file file.txt will now contain:

Hello!

Appended text.

This code opens file.txt in append mode ('a'), allowing new content to be added without overwriting existing data. It appends the text "\nAppended text.", adding it on a new line in the file.

Closing a File

The file.close() method is used to manually close a file after performing operations on it, ensuring system resources are freed.

Failing to close a file can lead to resource leaks or issues accessing the file elsewhere.

Syntax:

file.close()     #Close the file manually
