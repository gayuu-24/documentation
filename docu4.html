Functions in python

  • Functions are reusable blocks of code designed to perform specific tasks. They help in organizing code and avoiding repetition.
  • These concepts make Python functions flexible and powerful, allowing developers to write clean and efficient code.

Defining and Calling Functions

  • Defining a Function: Use the def keyword, followed by the function name and parentheses.
  • Calling a Function: Use the function name followed by parentheses.

Example:

def greet():                     # Defining a function

   print("Hello, World!")

greet()                            # Calling the function

The code defines a function named 'greet()' that contains a single statement to print "Hello, World!". Functions are reusable blocks of code that execute when called. Calling greet() executes the function, displaying the message "Hello, World!" on the output screen.

Output:

Hello, World!

Function Arguments and Return Values

  • Arguments: Functions can accept inputs (parameters) to work with dynamic data.
  • Return Values: Functions can return a result using the return statement.

Example:

def add_numbers(a, b):                    # Function with arguments

   return a + b

result = add_numbers(5, 7)             # Calling the function with arguments

print("Sum:", result)

The function add_numbers(a, b) takes two parameters, a and b, and returns their sum using the return a + b statement.

It is called with arguments 5 and 7, which are passed as a and b, respectively, producing the sum 12.

The result is stored in the variable result and printed as "Sum: 12" using the print function.

Output:

Sum: 12

Lambda (Anonymous) Functions

  • Lambda Functions: Short, single-expression functions defined using the lambda keyword.
  • Commonly used for short, simple tasks or as arguments in higher-order functions.

syntax:

lambda arguments: expression

Example:

x = lambda a: a + 10

print(x(5))

The lambda function x takes one argument a and returns a + 10.

When called with x(5), it adds 10 to 5 and prints the result 15.

Output:

15
