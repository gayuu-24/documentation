Classes and Objects

  • Python is an object oriented programming language.
  • Almost everything in Python is an object, with its properties and methods
  • A Class is like an object constructor, or a "blueprint" for creating objects

Creating a class:

To create a class, use the keyword class

class MyClass:

    x = 5

Creating a object:

  • We can use the class named MyClass to create objects

class MyClass:

    x = 5

obj = MyClass()

print(obj.x)

Output:

5

Self parameter

The self parameter represents the current object of the class and lets you access its attributes and methods.

It is automatically passed when calling a method on an object.

Example:

class MyClass:

   def show(self):

          print("This is a method of the class.")

obj = MyClass()

obj.show()

In this example, self refers to the object obj when calling the show() method.

Output:

This is a method of the class.

Object Methods

Objects can also contain methods. Methods in objects are functions that belong to the object.

Example:

class Dog:

    def bark(self):

      return "Woof!"

my_dog = Dog()

print(my_dog.bark())

The Dog class has a method bark that returns the string "Woof!".An object my_dog is created from the class, and calling my_dog.bark() prints "Woof!".

Output:

Woof!

Inheritance

  • Inheritance allows us to define a class that inherits all the methods and properties from another class.
  • Parent class is the class being inherited from, also called base class.
  • Child class is the class that inherits from another class, also called derived class.

Example:(Single Inheritance)

class dad():                              #parent class

   def phone(self):

      print("dads phone")

class son():                              #child class

   def lap(self):

       print("sons lap")

ram=son()

ram.lap()

Output:

Sons lap

Example:(Multiple Inheritance)

class dad():                              #parent class

   def phone(self):

      print("dads phone")

class mom():                              #child class

   def sweet(self):

      print("moms sweet")

class son(dad,mom):                  #child class

   def lap(self):

       print("sons lap")

ram=son()

ram.sweet()

ram.lap()

The above program explains, where the son class inherits from both dad and mom classes. When the object ram (an instance of son) calls sweet() and lap(), it executes methods from both the mom and son classes. This shows how the child class can access methods from multiple parent classes.

Output:

moms sweet

dads phone
