Python syntax

Python syntax can be executed by writing directly in the Command Line

>>>|print("hello world")

>>>|hello world

Python uses Indendation to define block of codes

if True:

   print("Python uses indendation!")

Output:

Python uses indendation!

Comments

Python has commenting capability for the purpose of in-code documentation.

Comments start with a #, and Python will render the rest of the line as a comment:

# this is a comment line

Variables and Data types

Variables are created by assigning a value using the = operator.

No need to declare the type explicitly (dynamic typing).

x = 10

y = 3.14

name = "john"

is_active = True

Common Data types

  • int : Integer Number (eg : 10, -12 )
  • float : Decimal Numbers (eg : 3.14, -0.11)
  • string : Text enclosed in quotes (eg : "Apple", "orange")
  • boolean : Logical values (True, False)

Input and Output

  • Input : Use input() to get user input (always returns a string)

name = input (" Enter your name: ")

Output:

Enter your name:

  • Output : Use print() to display the output

print(" The result is ", 24)

Output:

The result is 24

Basic Operations

  • Arithmetic operators

Arithmetic operators are used with numeric values to perform common mathematical operations

Operator Name Example
+ Addition x + y
- Subtraction x - y;
* Multiplication x * y
/ Division x / y
% Modulus x % y
** Exponential x * * y
// Floor Division x / / y

Example:

x = 2

y = 3

print(x + y)

print(x // y)

Output:

5

0
